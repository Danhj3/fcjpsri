# FCJPSRI Landing

Sitio web promocional para la Facultad de Ciencias Jurídicas, Políticas, Sociales y Relaciones Internacionales de la Universidad Autónoma Gabriel René Moreno.

## Características principales

- Identidad visual inspirada en los colores institucionales con gradientes, tarjetas vítreas y tipografía Manrope.
- Estructura integral con secciones sobre la facultad, oferta académica, investigación, autoridades, vida estudiantil, noticias, admisiones, alianzas y contacto.
- Oferta académica actualizada que incorpora la Licenciatura en Trabajo Social con descripción y beneficios diferenciados.
- Conmutador español/inglés que actualiza textos, metadatos, etiquetas accesibles y marcadores temporales.
- Animaciones suaves para contadores, elementos destacados y menú móvil con soporte para preferencias de movimiento reducido.
- Formulario de admisiones con validaciones básicas e indicadores de estado en ambos idiomas.
- Página auxiliar `defensas.html` con cronograma, recursos y acompañamiento para defensas de grado enlazada desde Vida Estudiantil y la navegación principal.

## Uso

1. Clonar el repositorio o descargar los archivos.
2. Abrir `index.html` en cualquier navegador moderno.
3. Interactuar con el menú, cambiar de idioma y recorrer la experiencia responsive.
4. Navegar a `defensas.html` desde el botón de Vida Estudiantil o directamente para consultar el calendario de defensas.

No se requieren dependencias adicionales ni build steps.
